# Logo Direction

这不是最终品牌手册，而是 Qoreon 第一轮可执行 Logo 方向。

## 品牌判断

Qoreon 的核心不是“聊天”，而是“控制层”。

品牌语义应同时体现：

- 人类意图进入系统
- Agent 团队被组织和调度
- 执行链路可控、可回看、可持续优化

因此第一轮 Logo 不建议走“对话气泡”或“机器人头像”路线，应该走：

- 系统感
- 控制感
- 执行路径感

## 推荐方向

### 方向 A：Q-Loop

含义：

- 用 `Q` 作为品牌唯一核心字母
- 圆环表示闭环与持续优化
- 尾部切口表示从意图进入执行
- 内部节点表示被调度的 Agent 或任务单元

适合：

- GitHub 头像
- 文档页角标
- 命令行启动页

### 方向 B：Signal Grid

含义：

- 用一个结构化网格表达通道与协同
- 用一条高亮路径表示被激活的执行链路

适合：

- 官网 Hero 背景
- 社交传播图
- 产品大图

## 首发建议

首发先采用 `方向 A：Q-Loop`。

原因：

- 识别成本最低
- 适合 GitHub 场景的小尺寸展示
- 能同时承接“Qoreon”品牌名和“控制层”语义

## 视觉基线

### 颜色

- `Core Carbon`: `#0f1720`
- `Signal Teal`: `#14b8a6`
- `Warm Sand`: `#f4efe6`
- `Alert Gold`: `#f2b84b`

### 风格

- 几何
- 精确
- 克制
- 不做科技蓝泛滥
- 不做紫色 AI 套路

### 字体方向

- 英文标题：偏理性几何无衬线
- 中文说明：中性无衬线
- 不建议圆润可爱风

## 禁止项

- 不要直接套机器人头
- 不要做霓虹紫 + 太空渐变
- 不要让 `Q` 难以辨认
- 不要把控制层语义做成普通聊天应用风格
